# 任务三说明

任务三走竞速路线 1 圈：

```text
AC 直线 -> CB 左弧线 -> BD 直线 -> DA 右弧线
```

入口：

- 按键：`B24`
- UART0：`03`
- 函数：`run_race_laps(1U)`

## 起步与航向

任务三启动时会触发一次 ST011 声光提示。当前验收版不再保留 Task10，任务三使用起步时固件建立的 JY62 相对零点作为航向基准。

摆车时应按当前调参约定放车，尽量减少人工初始角度误差。

## 关键参数

| 参数 | 含义 |
|---|---|
| `RACE_TASK3_AC_HEADING_TARGET_CDEG` | AC 直线目标航向 |
| `RACE_TASK3_BD_HEADING_TARGET_CDEG` | BD 直线目标航向 |
| `RACE_TASK3_AC_FORCE_TURN_COUNT` | 任务三 AC 强制入弯距离 |
| `RACE_AC_POINT_ARM_COUNT` | AC 点位检测使能距离 |
| `RACE_BD_POINT_ARM_COUNT` | BD 点位检测使能距离 |
| `RACE_BD_FORCE_TURN_COUNT` | BD 强制入弯距离 |
| `RACE_STRAIGHT_BASE_PWM` | 任务三直线段基础 PWM |
| `RACE_ARC_BASE_PWM` | 任务三弧线段基础 PWM |

任务三使用稳定参数，除非明确要联调任务三，不要跟随任务四高速调参一起修改。

## 日志

任务三和任务四共用 `RACE_*` 日志系统：

- `RACE_RAM_LOG_ENABLE`：RAM dump。
- `RACE_UART_LOG_ENABLE`：实时文本日志。

复盘时优先看每段 `RACE_SEG*` 摘要：

- `dist`：分段距离。
- `seen/first/last/span`：红外线出现情况。
- `yaw0/yaw1/ypr/herr`：航向变化和结尾误差。
- `rsn=1`：正常点位。
- `rsn=2`：距离强制。

## 验收前检查

1. 编译通过。
2. ready 文案只包含 `01..04`。
3. 任务三从 `03` 或 `B24` 启动。
4. 任务三跑稳后再动任务四高速参数。
