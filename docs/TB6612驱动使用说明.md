# TB6612 驱动使用说明

本文档对应当前工程里的 `BSP/inc/bsp_tb6612.h` 和 `BSP/src/bsp_tb6612.c`。

## 1. 当前接线

| 模块 | 信号 | 工程 IO | 用途 |
|---|---|---|---|
| TB6612 | PWMA | PA0 / A00 | 左轮 PWM |
| TB6612 | PWMB | PA1 / A01 | 右轮 PWM |
| TB6612 | AIN1 | PB0 / B00 | 左轮方向 1 |
| TB6612 | AIN2 | PB1 / B01 | 左轮方向 2 |
| TB6612 | BIN1 | PB8 / B08 | 右轮方向 1 |
| TB6612 | BIN2 | PB3 / B03 | 右轮方向 2 |
| TB6612 | STBY | PB7 / B07 | 驱动使能，高电平有效 |

当前 SysConfig 中 `PWMA` 使用 `TIMA0_CCP0`，`PWMB` 使用 `TIMA0_CCP1`。方向脚和 `STBY` 都属于 `GPIOB`。

## 2. 初始化顺序

在 `main.c` 里先调用 TI 自动生成的初始化函数，再初始化 TB6612：

```c
SYSCFG_DL_init();
TB6612_Init();
```

`SYSCFG_DL_init()` 会初始化时钟、GPIO、PWM、串口等外设。`TB6612_Init()` 会拉高 `STBY`，然后让电机进入刹车状态，避免上电后乱转。

## 3. 速度范围

驱动使用 `-999 ~ 999` 表示速度：

| 数值 | 含义 |
|---|---|
| 正数 | 正转 |
| 0 | PWM 为 0 |
| 负数 | 反转 |
| 999 | 当前配置下的最大 PWM 命令 |

如果输入超过范围，驱动会自动限幅到 `999`。

## 4. 常用函数

### TB6612_SetMotor()

控制单个电机。

```c
TB6612_SetMotor(TB6612_MOTOR_A, 300);   // 左轮正转
TB6612_SetMotor(TB6612_MOTOR_B, 300);   // 右轮正转
TB6612_SetMotor(TB6612_MOTOR_A, -200);  // 左轮反转
```

当前约定：`TB6612_MOTOR_A` 是左轮，`TB6612_MOTOR_B` 是右轮。

### TB6612_SetDifferential()

控制左右轮差速，后面做循迹控制主要用这个函数。

```c
TB6612_SetDifferential(250, 250);  // 直行
TB6612_SetDifferential(320, 150);  // 右转：左轮快，右轮慢
TB6612_SetDifferential(150, 320);  // 左转：左轮慢，右轮快
TB6612_SetDifferential(0, 0);      // PWM 输出为 0
```

### TB6612_Brake()

主动刹车。测试电机或调 PID 时建议用它停车。

```c
TB6612_Brake();
```

### TB6612_Coast()

滑行停止。它会关闭 PWM，同时把方向脚置为低电平，车会靠惯性滑一段。

```c
TB6612_Coast();
```

### TB6612_Enable() / TB6612_Disable()

控制 `STBY` 引脚。

```c
TB6612_Enable();   // STBY=1，允许驱动输出
TB6612_Disable();  // STBY=0，驱动待机
```

正常控制电机时不用反复手动调用 `TB6612_Enable()`，因为 `TB6612_Init()`、`TB6612_SetMotor()`、`TB6612_Brake()` 会自动使能。

## 5. 旧接口

为了兼容早期例程，驱动仍保留：

```c
AO_Control(1, 300);  // A 路正转
BO_Control(1, 300);  // B 路正转
AO_Control(0, 300);  // A 路反转
BO_Control(0, 300);  // B 路反转
TB6612_Motor_Stop(); // 等价于 TB6612_Brake()
```

后续写循迹控制时，推荐优先用 `TB6612_SetDifferential()`，因为它直接表达“左轮速度、右轮速度”，更适合小车。

## 6. 方向校准

如果调用：

```c
TB6612_SetDifferential(250, 250);
```

小车不是向前走，有两种处理方式：

1. 交换对应电机的两根输出线。
2. 修改 `BSP/inc/bsp_tb6612.h` 中的方向宏：

```c
#define TB6612_A_FORWARD_IN1 (1U)
#define TB6612_A_FORWARD_IN2 (0U)
#define TB6612_B_FORWARD_IN1 (0U)
#define TB6612_B_FORWARD_IN2 (1U)
```

A 路对应左轮，B 路对应右轮。只反一个轮子时，只改对应那一组。

## 7. 推荐测试流程

先把车架空，避免突然冲出去，然后运行 `main.c` 里的 demo：

```c
TB6612_SetDifferential(250, 250);
TB6612_Brake();
TB6612_SetDifferential(300, 140);
TB6612_Brake();
TB6612_SetDifferential(140, 300);
TB6612_Brake();
```

确认左轮、右轮、前进方向都对之后，再接八路红外循迹模块和陀螺仪。
